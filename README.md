# LinkSimplifier

## 项目简介
一款基于C# WPF .NET Framework 4.8开发的直链解析工具

## 支持平台
| 云盘         | 解析支持            | 官网链接                  |
|--------------|---------------------|--------------------------|
| QQ邮箱文件中转站 | 单文件              | [mail.qq.com](https://mail.qq.com) |
| 蓝奏云        | 单文件/文件夹       | [lanzou.com](https://www.lanzou.com) |

## 使用指南

### 基本操作
1. 复制云盘分享链接到输入框
2. 点击【开始解析】按钮获取结果

### 链接格式说明
| 使用场景       | 格式示例                                      | 参数说明                |
|----------------|---------------------------------------------|------------------------|
| 普通文件       | `https://xxx.com/xxxx`                      | 直接粘贴原始链接        |
| 加密文件       | `https://xxx.com/xxxx&pwd=1234`             | 追加`&pwd=密码`        |
| 普通文件夹     | `https://xxx.com/xxxx&folder`               | 追加`&folder`          |
| 加密文件夹     | `https://xxx.com/xxxx&pwd=1234&folder`      | 同时追加密码和folder参数|
