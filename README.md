# LinkSimplifier

## 项目简介

LinkSimplifier 是一款专注于云盘直链解析的工具。当前提供基于 C# WPF .NET Framework 4.8 开发的桌面客户端，同时正在开发基于 Cloudflare Pages 的 Web 版本，实现跨平台、无需安装的便捷体验。

## 核心功能

- ⚡ **快速直链获取**：一键解析获取可直接下载的直链地址
- 🔐 **加密链接支持**：支持带密码的分享链接解析
- 📁 **文件夹解析**：支持解析文件夹

## 支持平台
| 云盘         | 解析支持            | 官网链接                  |
|--------------|---------------------|--------------------------|
| QQ邮箱文件中转站 | 单文件              | [mail.qq.com](https://mail.qq.com) |
| 蓝奏云        | 单文件/文件夹       | [lanzou.com](https://www.lanzou.com) |

## 使用指南

### 基本操作
1. 复制云盘分享链接到输入框
2. 点击【开始解析】按钮获取结果

### 链接格式说明
| 使用场景       | 格式示例                                      | 参数说明                |
|----------------|---------------------------------------------|------------------------|
| 普通文件       | `https://xxx.com/xxxx`                      | 直接粘贴原始链接        |
| 加密文件       | `https://xxx.com/xxxx&pwd=1234`             | 追加`&pwd=密码`        |
| 普通文件夹     | `https://xxx.com/xxxx&folder`               | 追加`&folder`          |
| 加密文件夹     | `https://xxx.com/xxxx&pwd=1234&folder`      | 同时追加密码和folder参数|
